# LSTM风电预测模型说明.md — CNN-LSTM Two-Step Multi-Output

## 一、模型架构（CNN-LSTM 混合网络）

本模型使用 **CNN-LSTM 混合神经网络**，共 5 层：

```
Input: (batch, 168h, 7 features)
    │
    ▼ 第 1 层 — Conv1D(filters=64, k=3, p=1, ReLU)
    │  作用：提取局部日周期特征（24h 周期性的低维表达）
    │  输出: (batch, 64, 168)
    │
    ▼ 第 2 层 — Conv1D(filters=128, k=3, p=1, ReLU)
    │  作用：提取短时波动特征（爬坡事件、风速骤变）
    │  输出: (batch, 128, 168)
    │
    ▼ 第 3 层 — MaxPool1D(pool_size=2)
    │  作用：降维，减少 LSTM 计算量，强化重要特征
    │  输出: (batch, 128, 84)
    │
    ▼ 第 4 层 — LSTM(hidden=100, num_layers=1)
    │  作用：捕捉 7 天序列中的长期时间依赖
    │  取最后时间步隐藏状态
    │  输出: (batch, 100)
    │
    ▼ 第 5 层 — Dense(50, ReLU) → Dropout(0.2) → Dense(24)
    │  作用：非线性映射 → 多步输出 24 小时预测
    │  输出: (batch, 24)
```

### 参数量概览

| 层        | 类型          | 参数量          |
| -------- | ----------- | ------------ |
| Conv1D-1 | 7→64, k=3   | ~1,408       |
| Conv1D-2 | 64→128, k=3 | ~24,704      |
| LSTM     | 128→100     | ~91,600      |
| FC-1     | 100→50      | ~5,050       |
| FC-2     | 50→24       | ~1,224       |
| **合计**   |             | **~124,000** |

### 风速预测（7 维输入特征）

```
wind_speed_100m, u100, v100, air_density, t2m,
hour_sin, hour_cos
```

### 空气密度预测（8 维输入特征）

```
air_density, t2m, sp, wind_speed_100m, u100, v100,
hour_sin, hour_cos
```

## 二、预测逻辑（Two-Step 两步法 × 24h 多步输出）

### 流程图

```
ERA5 数据 → wind_data.csv
                    │
       ┌────────────┴────────────┐
       │                         │
  Step 1a                      Step 1b
  CNN-LSTM 风速预测             CNN-LSTM 密度预测
  输入: 168h × 7特征            输入: 168h × 8特征
  输出: 24个风速值              输出: 24个密度值
  [t+1, t+2, ..., t+24]       [t+1, t+2, ..., t+24]
       │                         │
       └────────────┬────────────┘
                    │
              Step 2 — 物理公式
              Vestas V90 功率曲线
              + 对数风廓线修正 (100m→90m)
              + 空气密度修正
              逐小时计算 24 个功率值
                    │
                    ▼
              24h 风电功率曲线
```

### Step 1a: CNN-LSTM 风速预测

1. **输入序列**：过去 **168 小时（7 天）**的逐小时气象数据，构成 (168, 7) 张量
2. **CNN 特征提取**：2 层 Conv1D 提取局部日周期和短时波动模式
3. **MaxPool 降维**：序列从 168 → 84 步，保留关键信息
4. **LSTM 时序建模**：1 层 LSTM(100) 学习 7 天序列中的长期依赖
5. **多步输出**：全连接层直接输出 **24 个值** —— t+1 到 t+24 的风速

### Step 1b: CNN-LSTM 空气密度预测

- 结构同上，特征维 8，目标为 24h 空气密度

### Step 2: 物理公式计算风电功率

```
v_hub[t+h] = v_100m[t+h] × ln(90/0.03) / ln(100/0.03)   风廓线修正
P[t+h] = Vestas_V90(v_hub[t+h]) × (ρ[t+h] / 1.225)      功率曲线 + 密度修正
```

## 五、关键改进分析

### 相比纯 LSTM 的提升

1. **风速 R2 从 0.019 → 0.216（11 倍提升）**
   
   - CNN 层成功提取了风速的局部模式，弥补了纯 LSTM 在强随机信号上的不足

2. **功率 NMAE 从 16.4% → 14.0%**
   
   - 风速预测精度的提升直接转化为功率预测精度

3. **空气密度 R2 从 0.87 → 0.92**
   
   - 7 天长序列提供了更丰富的大气状态演化信息

4. **短时预测极准（h=1 功率 MAE 仅 121 kW）**
   
   - 多步输出让模型学习到短期预报的高可预测性

5. **R2 均为正（纯 LSTM 功率 R2=-0.14）**
   
   - CNN-LSTM 的预测方差已小于实际方差，超越了"预测均值"这个朴素基线

### 待优化方向

- 风速长时预测（h>18）仍较弱 → 可尝试注意力机制
- 模型仍有过拟合倾向（epoch 4 最佳）→ 增大 dropout 或添加更多正则化
- 可尝试 BiLSTM 捕获双向时序依赖
